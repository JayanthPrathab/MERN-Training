The 2 types of VCS include
  1. Centrlaized VCS
  2. Distributive VCS
Git Command Function
git init Initialises an empty local repository
git add FILE_NAME Adds the selected files/folder name to the local repository
git commit -m "COMMIT_MESSAGE" commits the files to the staged files
git push -u origin BRANCH NAME Pushes the files to the global repository ie. to the repository
git remote add origin URL When we need to change the repository, we need to change the repository URL
git pull When we commit changes directly on GitHub(Global repository), we use this command to affect/save changes to the files in the local repository
